### Web Start
- Go to  `cordovaCovidPulse`
- `npm install`
- `npm start`


### Mobile setup
- Install `Android SDK` & `Emulator` 
- Setup Cordova 
```
- npm i -g cordova
- cordova platform add platform
- cordova build android
- cordova emulate android (Emulator)
- cordova run androud (Run in Android)
```
